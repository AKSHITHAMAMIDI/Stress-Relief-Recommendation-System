# 🌿 Unwind — Stress Relief Recommendation System

A pure-Python web app (built with [Streamlit](https://streamlit.io)) that estimates
your current stress level from a few lifestyle questions and recommends
movies, videos, and games to help you relax — personalized to how stressed
you are and what mood you're in.

No separate frontend/backend, no JavaScript, no server setup — just Python.

---

## How it works

1. **`train_model.py`** — generates a synthetic-but-realistic lifestyle
   dataset (sleep, work hours, screen time, activity, mood, social
   connection, caffeine, meditation) and trains a `RandomForestClassifier`
   from scikit-learn to predict stress level (**Low / Medium / High**).
   Saves the trained model + scaler to disk.
2. **`recommendations.py`** — a curated database of movies, videos, and
   games, each tagged with the stress levels and "mood" (calming,
   uplifting, comforting, reflective, energizing) it suits. Contains the
   matching logic.
3. **`app.py`** — the Streamlit web app. Collects your answers via sliders,
   feeds them through the trained model, shows your stress level with a
   confidence chart, and displays personalized recommendations in tabs.

## Setup & run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the model (only needed once — creates .pkl files)
python train_model.py

# 3. Launch the app
streamlit run app.py
```

Streamlit will open the app automatically in your browser at
`http://localhost:8501`.

## Customizing

- **Add more recommendations:** edit the `MOVIES`, `VIDEOS`, or `GAMES`
  lists in `recommendations.py`. Each entry needs a `title`, `genre`,
  `levels` (which stress levels it fits), and a `mood` tag.
- **Change the stress model:** adjust the feature weights or add new
  features inside `generate_synthetic_dataset()` in `train_model.py`, then
  re-run it to retrain.
- **Use real survey data instead of synthetic data:** replace
  `generate_synthetic_dataset()` with a function that loads your own CSV
  (keep the same column names listed in `FEATURES`).

## Note

This app gives general relaxation suggestions based on self-reported
lifestyle data — it is not a diagnostic or medical tool. If stress is
significantly affecting your life, please consider speaking with a doctor
or mental health professional.
